# dynamic-template-addon

> Vue.js, Vuetify, Storybook

## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# serve Storybook with hot reload at localhost:6006
$ npm run storybook
```
